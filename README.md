# FitTrack

FitTrack е уеб приложение за проследяване на тренировки и хранителен режим.  
Потребителите могат да добавят, редактират, преглеждат и изтриват записи за тренировки и хранения.

---

## Основни функционалности

### Тренировки
- Добавяне на нова тренировка
- Преглед на всички тренировки
- Редактиране на съществуващи тренировки
- Изтриване на тренировки

### Хранене
- Добавяне на ново хранене
- Преглед на всички хранения
- Изтриване на хранения

### База данни
- Съхранение на данни в MySQL
- Релационни таблици за потребители, тренировки и хранения

---

## Технологичен стек

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (Flask)  
- **Database:** MySQL  
- **Tools:** Git, GitHub, Visual Studio Code  
- **External API:** Nutritionix (в процес на одобрение)

---

## Външен API

В момента достъпът до API е в процес на одобрение.  
След получаване на API ключове ще бъде добавено автоматично извличане на хранителна информация.

## Инструкции за стартиране

### 1. Клониране на проекта
```bash
git clone https://github.com/your-username/fittrack.git
cd fittrack

### 2. Инсталиране на зависимости
pip install flask flask-cors mysql-connector-python

### 3. Настройка на база данни
Стартирайте MySQL (например чрез XAMPP)
Създайте база данни с име: fittrack
Импортирайте SQL файла от папка database

### 4. Стартиране на приложението
python app.py

### 5. Отваряне в браузър
Отворете:
http://127.0.0.1:5000
